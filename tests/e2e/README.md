# End-to-end tests

**Post-MVP — intentionally empty in Unit 01.**

This directory is forward-declared so the convention exists before it is needed.
Full-stack e2e specs (browser-driven walkthroughs of the library, book reading,
multiplayer presence) land here after the hero features exist. They are not part
of the MVP unit-only test gate (C5), and CI does not run them yet.
